# Twitter clone  
 simple twitter clone written in Go with [redis](https://redis.io/)
 
## Features
1. signUp and logIn 
2. follow users
3. post tweets
4. home page 
5. timeline

## Dependencies
name     | repo
------------- | -------------
  go redis    | https://github.com/go-redis/redis
  httprouter  | https://github.com/julienschmidt/httprouter
  go.uuid     | https://github.com/satori/go.uuid
